# Sneaker Store Frontend

This is a Next.js + TypeScript e-commerce frontend for Sneaker Store.

## Features
- Product listing and details
- Cart and checkout
- Order success and tracking
- User authentication (login/signup)
- Dashboard (profile, orders)
- Zustand for state management
- Tailwind CSS for styling

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Folder Structure
- `app/` - Main app directory (pages, components)
- `lib/` - Utilities and Zustand store
- `public/` - Static assets (images, etc.)

---

For backend/API integration, see the backend folder.
